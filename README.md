# SQL-Bikestore-Insights

## Project Overview

SQL-Bikestore-Insights is a comprehensive analytics project built on SQL, focusing on turning raw bike store sales data into actionable business insights. The project delivers an interactive sales dashboard and KPI visualizations, helping retail stakeholders optimize revenue, inventory, and strategy decisions.

---

## Features

- **Interactive Dashboard:** Visualizes total sales, revenue, discounts, quantity, and delivery metrics.
- **Category & Brand Analysis:** Summarizes top-performing categories (Mountain, Road) and brands (Trek, Electra, Surly).
- **Geographic Insights:** Statewise sales mapped for regional benchmarking.
- **Automated Reporting:** SQL queries enable rapid, repeatable business reporting.

---

### Technology Used
- MS SQL Server 
- BI visualization tool (Power BI)
